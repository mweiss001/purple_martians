 Items
Overview
Overview Items are one of the 5 basic types of objects that interact in the game. These are all types of items: - start - exit - door - key - switch - spring - bonus - bomb - rocket - mine - msg - timer - trigger - gate They all live in 2 arrays that can hold 500 items. class mwItem { int item[500][16]; // item ints float itemf[500][4]; // item floats Each item has 16 ints for data and 4 floats for position and momentum. when the game loop is running, these functions bring them to life: void move_items(); void proc_item_collision(int p, int i); void draw_items(void);
