PURPLE MARTIANS

by Michael David Weiss

Download

Download the latest release from Github:

https://github.com/mweiss001/purple_martians/releases
Installation

- download the windows or linux version
- extract to a folder and run the executable

There is nothing to install.
The game is completely self contained within its folder.
Nothing is stored or changed outside that folder.

Windows Specific Instructions

Download and extract the contents of the zip file.

In the extracted folder double click on 'pm.exe'.

If windows complains about an unrecognized app, click 'More info' then 'Run anyway'.

Linux Specific Instructions

Download and extract the contents of the tar.gz file.

Open a terminal in the extracted folder and type './pm'.
