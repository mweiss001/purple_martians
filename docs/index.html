PURPLE MARTIANS

by Michael David Weiss

Description

A free 2D platform style video game, with a mix of puzzle solving and shooting.

Key Features

- Single player or multiplayer with up to 8 players

- Over 70 fun and challenging levels

- Fully featured level editor to create new levels

- Completely free

- Made entirely with free software

- Works identically in linux and windows
