Description Purple Martians is free 2D platform style game, with a mix of puzzle solving and shoot-em-up. Features: - over 70 fun and challenging levels - single player or networked multiplayer with up to eight players - an excellent level editor to edit and create new levels - works identically across windows and linux - very smooth graphics - dynamically resizable display, moveable, zoomable during gameplay - works with screen resolutions from 320x240 to 4K and higher Videos
