PURPLE MARTIANS

by Michael David Weiss

Multiplayer Overview

Up to 8 players can play together.

Players can cooperate, or compete.

There is one server and up to 7 clients can join.

Clients need the server's IP or hostname.

Official Purple Martians Server

The official Purple Martians server is:

purplemartians.org

Host your own server in a LAN

Pick one computer to be the server and determine its local IP address.

On the clients, set the server IP to that IP.

Host your own server over the internet

Determine your external IP address.

On your router, set up a port forwarding rule to allow incoming UDP traffic on port 24785 to be forwarded to your server's local IP address.

On the remote clients, set the server IP to the external IP address.
